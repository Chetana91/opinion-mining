Instructions:

1. Place the tranining and test file in the same folder as the code file

2. Run the file with the sheet name as command line argument
	2.1 For Obama
	python senti_analysys.py Obama

	2.2 For Romney
	python senti_analysis.py Romney

Dependencies:

The program has the following dependencies:
	1. Openpyxl (2.3.0) for reading from excel file
	2. nltk package
	3. BeautifulSoup
	4. nltk PorterStemmer
	5. nltk metrics
	6. nltk RegexpTokenizer
